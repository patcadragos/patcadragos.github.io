Project Overview

The FOLLICE BOX is an innovative autonomous follow-me cooler designed for outdoor environments. It combines a 50L insulated storage unit with a 24V motorized differential drive system, advanced obstacle detection, and up to 3 hours of runtime. Built for portability and efficiency, this project showcases autonomous robotics in a practical, consumer-friendly application.

Mechanical Design

Chassis & Structure

The Follice Box features a sturdy 24V motorized differential drive chassis designed for autonomous movement across varied terrain. The chassis supports the full 50L insulated storage unit with balanced weight distribution for optimal stability.

Drive System

Differential drive mechanism enables precise turning and smooth acceleration. Two independently controlled motors provide responsive steering for obstacle avoidance and user tracking capabilities.

Insulation & Storage

A high-quality 50L insulated storage compartment maintains temperature integrity for beverages and provisions. The design prioritizes both capacity and portability for outdoor use.

Electronics & Control Systems

ESP32 Cytron MDDS30 Li-ion 7S ToF Sensor 24V Motor Differential Drive

Autonomous Navigation

  • ESP32 Microcontroller: Core processing unit for autonomous decision-making
  • ToF (Time-of-Flight) Sensor: Precise obstacle detection and distance measurement
  • Obstacle Avoidance: Real-time sensor fusion for safe autonomous operation
  • User Tracking: Follows operator at adjustable distance

Power System

  • Li-ion 7S Battery Pack: High-capacity energy storage for extended runtime
  • 24V Motor System: Reliable, field-tested motor technology
  • Cytron MDDS30 Motor Driver: Professional-grade motor control with safety features
